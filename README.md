# 🎓 Tuple Student Manager (Python)

A simple and interactive **CLI-based Student Record Manager** built using **Python Tuples**.
This project helps manage student data efficiently with a clean menu-driven interface.

---

## 🚀 Live Repository

👉 https://github.com/rishii-in/tuple-student-manager-python

---

## ✨ Features

* ➕ Add Student Records
* 📋 View All Students
* 🏆 Find Topper
* 📊 Calculate Average Marks
* 🔍 Search Student by Name
* 🚪 Exit Program

---

## 🧠 Concepts Used

* Tuples (Core Concept)
* Tuple Operations
* Loops (`for`, `while`)
* Conditional Statements
* Exception Handling
